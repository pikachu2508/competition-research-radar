# 竞赛研究雷达

围绕“非平衡电桥测铜电阻温度系数”建立的个人学术信源系统。

## 使用

- 双击 `启动研究雷达.cmd`：检查更新、发送强相关飞书提醒并打开本地页面。
- 双击 `安装每日自动更新.cmd`：设置每天 08:00 检查、20:00 飞书晚间精选，并在每周日 20:10 生成周报。
- 本地页面：`output/index.html`。

首次运行会导入现有内容作为资料基线，不会把历史论文集中发送到飞书。以后只有新出现的强相关内容才会提醒。

## 当前范围

- 知网：《物理实验》《大学物理实验》《大学物理》《计量科学与技术》《计量学报》《传感技术学报》《传感器与微系统》
- 国际期刊：Physics Education、European Journal of Physics、Measurement Science and Technology、Metrologia
- 权威动态：NIST Metrology、NIST Physics

## 数据位置

- 本地资料库：`data/radar.sqlite3`
- 页面：`output/index.html`
- 最近运行记录：`logs/latest.log`
- 每周精选：`reports/`
- 信源和筛选规则：`config/`

系统只保存信源公开提供的标题、作者、摘要和原文链接，不自动下载全文。

知网 RSS 可自动提供作者和期刊日期。作者单位、DOI、关键词、基金和“知网平台首次公开时间”来自论文详情页；由于知网可能要求安全验证，这些字段采用成功后缓存的方式补充，抓取失败不会影响日常 RSS 更新。期刊日期与知网首次公开时间会分开显示。

## 云端运行

GitHub Actions 以北京时间每天 09:00 为目标发送早报，09:10 和 09:30 自动补偿。同一天只发送一次；即使没有新论文，也会发送“今日无新增”并推荐一篇核心资料。GitHub Pages 同步发布最新雷达页面。

仓库 Secrets：`LARK_APP_ID`、`LARK_APP_SECRET`、`LARK_CHAT_ID`。密钥不得写入代码或日志。
